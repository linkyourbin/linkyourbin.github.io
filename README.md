# LinkYourBin

这是我的个人技术网站，专注于嵌入式开发和AI技术。

## 网站结构

- `index.html` - 主页
- `assets/` - 静态资源文件夹
  - `css/style.css` - 样式文件
  - `js/script.js` - JavaScript文件
  - `images/` - 图片资源
- `pages/` - 页面文件夹
  - `chips.html` - 芯片页面
  - `projects.html` - 项目页面
  - `about.html` - 关于页面
  - `contact.html` - 联系页面
  - `404.html` - 404页面

## 技术栈

- HTML5
- CSS3
- JavaScript
- Font Awesome 图标库

## 部署

该网站通过 GitHub Pages 部署。
