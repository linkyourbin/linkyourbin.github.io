# Embedded Developer Portfolio

This is a personal portfolio website for an embedded systems developer, showcasing projects and expertise with STM32, ESP32, RP2040, RP235x microcontrollers and programming in C, Rust, and Python.

## Features

- Responsive design with mobile support
- Project showcase section
- Microcontroller expertise section
- About me and contact information
-粉蓝白 (Pink-Blue-White) color scheme

## Deployment

This website is designed for deployment on GitHub Pages. To deploy:

1. Fork this repository
2. Rename the repository to `yourusername.github.io`
3. Push your changes to the `main` branch
4. Enable GitHub Pages in the repository settings

## Structure

- `index.html` - Homepage
- `pages/` - Additional pages (about, projects, chips, contact, 404)
- `assets/css/` - CSS stylesheets
- `assets/js/` - JavaScript files
- `assets/images/` - Image files

## Customization

1. Replace `logo.png` in `assets/images/` with your own logo
2. Update text content to reflect your personal information
3. Replace placeholder images with your own project photos
4. Modify the color scheme in the Tailwind configuration if needed
