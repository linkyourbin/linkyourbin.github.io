STM32环境监测系统

STM32 传感器 C语言

项目概述

这是一个基于STM32F407的环境监测系统，能够实时监测温度、湿度、空气质量和光照强度，并通过OLED屏幕显示数据。系统还支持数据存储和USB数据导出功能。

技术细节

  • 主控制器：STM32F407IGH6
  • 传感器：DHT11（温湿度）、MQ-135（空气质量）、BH1750（光照）
  • 显示：0.96寸OLED显示屏（I2C接口）
  • 存储：SD卡模块（SPI接口）
  • 通信：USB虚拟串口
  • 开发环境：STM32CubeIDE

功能特点

  • 实时数据采集与显示
  • 数据定时存储到SD卡
  • USB接口数据导出与配置
  • 超阈值报警功能
  • 低功耗模式支持

ESP32智能家居控制系统

ESP32 物联网 C语言 Python

项目概述

这是一个基于ESP32的智能家居控制系统，支持灯光、窗帘和温度调节。系统通过Wi-Fi连接互联网，可通过手机APP或语音助手（如Alexa、Google Assistant）进行控制。

技术细节

  • 主控制器：ESP32-WROOM-32
  • 通信：Wi-Fi、蓝牙
  • 控制方式：继电器、PWM调光
  • 手机APP：基于Blynk平台
  • 语音控制：集成Amazon Alexa
  • 开发环境：PlatformIO

功能特点

  • 远程控制家电设备
  • 定时任务与场景模式
  • 语音控制集成
  • 能耗监测
  • OTA固件升级

Rp2040便携式游戏开发板

Rp2040 游戏开发 C语言 MicroPython

项目概述

这是一个基于Raspberry Pi Pico（Rp2040）的便携式游戏开发板，配备1.8英寸TFT显示屏和按键矩阵。该开发板支持C语言和MicroPython编程，适合学习嵌入式系统和游戏开发。

技术细节

  • 主控制器：Rp2040（Raspberry Pi Pico）
  • 显示：1.8英寸TFT显示屏（160x128像素）
  • 输入：8按键矩阵
  • 电源：3.7V锂电池，支持USB充电
  • 扩展：I2C和SPI接口
  • 开发环境：Visual Studio Code + Pico SDK

功能特点

  • 支持C和MicroPython开发
  • 内置多款示例游戏（贪吃蛇、俄罗斯方块等）
  • 低功耗设计，续航可达8小时
  • 可扩展传感器和其他外设
  • 开源硬件设计

RP235x自主导航机器人

RP235x 机器人 Rust

项目概述

这是一个基于最新RP2353微控制器的自主导航机器人项目。该机器人能够避开障碍物，规划路径，并通过手机APP进行远程控制。项目使用Rust语言开发，展示了RP235x系列的高性能和安全特性。

技术细节

  • 主控制器：RP2353（双核Cortex-M33）
  • 传感器：超声波测距模块、红外避障传感器
  • 运动：直流减速电机+编码器
  • 通信：Wi-Fi和蓝牙
  • 电源：7.4V锂电池
  • 开发语言：Rust

功能特点

  • 自主避障和路径规划
  • 手机APP远程控制与监控
  • OTA固件升级
  • 基于Rust的内存安全设计
  • 低功耗模式延长续航
