STM32 Series

STMicroelectronics' STM32 family is one of the most versatile microcontroller lines available, featuring ARM Cortex-M0, M0+, M3, M4, M7, and M33 cores. I've worked extensively with various STM32 models including the F1, F4, L0, and H7 series.

Key Features & Experience:

Advanced timer configurations for motor control
DMA for efficient peripheral data transfer
Low-power modes for battery-operated devices
CMSIS and HAL libraries proficiency
RTOS implementation (FreeRTOS, Zephyr)
USB device and host implementations

ESP32 Series

Espressif's ESP32 microcontrollers have revolutionized IoT development with their built-in WiFi and Bluetooth capabilities. I've worked with various ESP32 variants including the ESP32-C3, ESP32-S3, and ESP32-C6.

Key Features & Experience:

WiFi (802.11 b/g/n) and Bluetooth Classic/BLE
IoT protocol implementation (MQTT, CoAP, HTTP)
OTA (Over-The-Air) firmware updates
MicroPython and Arduino framework experience
Secure communication with TLS/SSL
Power management for battery efficiency
View ESP32 Projects

RP2040 & RP235x

Raspberry Pi's entry into microcontrollers with the RP2040 and its successor RP235x offer unique capabilities with their PIO (Programmable I/O) state machines and powerful processing cores.

Key Features & Experience:

PIO programming for custom peripherals
Rust embedded development with embassy
RP235x's Cortex-M33 and DSP capabilities
UF2 bootloader and easy programming
W5500 Ethernet and connectivity expansions
Low-cost development with Raspberry Pi Pico
